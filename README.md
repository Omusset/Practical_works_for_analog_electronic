# Practical_works_for_analog_electronic
Files to build your own setup for analog electronic experiences
Teaching basic analogue electronics has become increasingly challenging in undergraduate physics curricula, particularly for students not specialising in electronics. While numerical simulation tools are widely used, students’ engagement with hands-on experimental work has significantly declined, as have fundamental practical skills such as circuit wiring and elementary electrical analysis. However, in any physical measurement chain, analogue signal conditioning remains an essential step prior to digitisation, even when using widely adopted educational platforms such as Arduino or Raspberry Pi systems. We present a compact and low-cost experimental setup designed for third-year undergraduate laboratories that integrates three printed circuit boards covering the complete measurement chain: (i) AC-to-DC conversion and voltage regulation, (ii) operational amplifier-based analogue functions, and (iii) passive and active filters combined with a low-frequency white-noise generator. The system is developed following a do-it-yourself (DIY) approach, including the use of fused deposition modelling (FDM) 3D printing for the mechanical housing. The solution is straightforward to implement using widely available components and requires only standard laboratory instrumentation. The white-noise source, used together with the fast Fourier transform (FFT) function of a standard digital oscilloscope, enables experimental reconstruction of Bode-like frequency responses and provides a direct link between time-domain and frequency-domain analysis. The setup is complemented by simulations performed with LTspice. All circuit schematics, PCB files and simulation resources are freely available in an open-access repository hosted on GitHub. This approach strengthens students’ conceptual understanding of analogue signal processing while maintaining quantitative agreement between theory and experiment.
